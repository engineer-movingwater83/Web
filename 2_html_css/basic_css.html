Tag, Class, ID in HTML with CSS

HTML can has not only Tag, but also Class and ID.

For main reason why make Class and ID are usually that can be used in CSS.

  1. Tag Selector

    HTML : 

     <p> This is p tag. </p>

    CSS : 

    p { 
       background-color: #09C; 
    }

  2. Class Selector

    HTML : 

     <p class="myClass"> This is Class </p>

    CSS : 

    .myClass { 
       text-align: center; 
    }

    Here is the real example.

    This is Class

  3. ID Selector

    HTML : 

     <p id="myId"> This is Id </p>

    CSS : 

    #myId { 
       background-color: #09C; 
    }

    Here is the real example.

    This is Id

  4. tag, CSS, ID Selector

    HTML : 

     <p class="secondClass" id="secondId"> This is p tag and secondClass and secondId </p>
     <p class="secondClass"> This is p tag and secondClass</p>
     <p> This is p tag </p>

    CSS : 

    p { 
       text-decoration: underline; 
    }
    .secondClass { 
       background-color: rgb(17, 209, 59); 
    }
    #secondId { 
       background-color:rgb(206, 228, 15); 
    }

    Here is the real example.

    This is p tag and secondClass and secondId

    This is p tag and secondClass

    This is "p tag"

    As a result, you can know that priority is Id first and Class, and tag is the least.
